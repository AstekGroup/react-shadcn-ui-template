# PROJECT_VIBE_CONF

## Type de Projet
**Template / Sandbox**

Ce projet est un **template** conçu pour être réutilisé dans différents contextes. Il sert de base pour démarrer rapidement des projets React avec Shadcn UI.

## Classification

- **Type**: Template
- **Besoin DEVPLAN**: ✅ Oui (voir DEVPLAN.md - déjà complété)
- **Besoin ADR**: ❌ Non (template de référence)
- **Documentation complète**: ✅ Oui

## Description

Template React moderne avec :
- React 19.2.0
- Vite 7.1.12
- TypeScript 5.9.3 (strict mode)
- Shadcn UI (composants)
- Tailwind CSS 4.1.16
- Dark mode (next-themes)
- Dashboard complet pré-configuré

## Objectif

Fournir un point de départ robuste et prêt pour la production pour les applications React, avec :
- Composants UI accessibles et personnalisables
- Configuration TypeScript stricte
- Support dark mode natif
- Dashboard de démonstration
- Documentation complète pour développeurs et agents IA

## Utilisation

Ce template peut être :
1. Cloné directement dans un nouveau projet
2. Intégré dans un monorepo
3. Utilisé comme référence pour de nouveaux projets
